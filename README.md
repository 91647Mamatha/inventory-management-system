# Inventech - Inventory Management System

## 📌 Overview

Inventech is a web-based Inventory Management System developed to efficiently manage products, customers, suppliers, and orders. It helps businesses track stock, manage transactions, and maintain organized records in a centralized system.

---

## 🚀 Features

* 🔐 Secure authentication and authorization
* 📦 Product management (Add, update, delete, view)
* 👥 Customer management
* 🚚 Supplier management
* 🧾 Order management
* 📊 Admin dashboard for overview and control
* 🔎 Search and filter functionality

---

## 🛠️ Tech Stack

* Backend: Java, Spring Boot, Spring Security
* Database: PostgreSQL
* ORM: JPA (Hibernate)
* Frontend: Thymeleaf
* Build Tool: Maven

## ⚙️ Setup & Installation

1. Clone the repository:

```
git clone https://github.com/91647Mamatha/inventory-management-system.git
```

2. Navigate to project folder:

```
cd inventory-management-system
```

3. Configure database in `application.properties`

4. Run the application:

```
mvn spring-boot:run
```

5. Open in browser:

```
http://localhost:8080
```
